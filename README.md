# AI Engineering Starter Kit

> Build software **with AI**, not just **using AI**.

A production-ready starter kit for AI-native software engineering.

This repository is **not** another prompt collection.

It is a complete engineering workflow designed for AI coding agents such as Cursor, Claude Code, Codex, Gemini CLI, OpenHands, and future AI development tools.

The goal is simple:

> **Help AI become a reliable engineering teammate instead of a code generator.**

---

# Why This Project Exists

Most AI coding today follows the same pattern:

```
Idea
    ↓
Prompt
    ↓
Generate Code
```

This works for small demos.

It breaks down for real software projects.

As projects grow, AI begins to:

* duplicate implementations
* introduce inconsistent architectures
* forget previous decisions
* rewrite unrelated modules
* invent APIs
* drift away from the original design

The problem is not the model.

The problem is that software engineering knowledge exists only in the developer's head.

This project moves that knowledge into structured documents that every AI agent can understand.

---

# Our Philosophy

```
Vision

    ↓
```
